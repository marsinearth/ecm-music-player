# ECM Music Player for samples

`create-react-app --template cra-template-pwa-typescript`를 이용한 Music Player PWA 앱.

## Summary

[ECM Records](https://ecmrecords.com/)에서 제공하는 무료 음원 샘플들을 한곳에 편히 들을 수 있도록 모아놓은 플레이어 앱.
